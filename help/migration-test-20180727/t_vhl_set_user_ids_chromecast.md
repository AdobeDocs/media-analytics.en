# User Identifier

Set and get the unique user ID on the ADBMobile SDK as follows:

>1. **Set:**
>
>   ```
>   ADBMobile().config.setUserIdentifer("app-generated-unique-id");
>   ```
>
>1. **Get:**
>
>   ```
>   vid = ADBMobile().config.getUserIdentifer();
>   ```
>
