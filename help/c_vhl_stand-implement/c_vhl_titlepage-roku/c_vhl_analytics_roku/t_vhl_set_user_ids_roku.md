# User Identifier

Set and get the unique user ID on the ADBMobile SDK as follows:

>1. **Set:**
>
>   ```
>   ADBMobile().setUserIdentifer("app-generated-unique-id")
>   ```
>
>1. **Get:**
>
>   ```
>   vid&amp;nbsp;=&amp;nbsp;ADBMobile().userIdentifer()
>   ```
>
